# Python File Management and Data Processing Utility

This is a group project for **INTE 472: Scripting Languages**.

## Project Purpose

[cite_start]This utility assists users in creating, backing up, organizing, logging, and analyzing data files automatically using Python[cite: 6].

## How to Run

1.  Clone the repository.
2.  Ensure you have Python 3 installed.
3.  Run the main script:
    ```bash
    python file_manager.py
    ```
4.  Run the reporting script:
    ```bash
    python students_report.py
    ```
